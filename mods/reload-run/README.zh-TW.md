# reload-run

[English](README.md) | [简体中文](README.zh-CN.md) | 繁體中文

由 **Jayden Liang** 開發的 [Slay the Spire 2](https://store.steampowered.com/app/2868840/Slay_the_Spire_2/) Mod。

## 功能

### F5 — 重新載入當前房間

在對戰中按 **F5**，立即從當前房間的入口存檔重新開始。

- **單人模式** — 原地重新載入，無需返回主選單
- **多人（主機）** — 返回主選單並自動以主機身份恢復已儲存的多人對戰
- **多人（其他玩家）** — 返回主選單並自動開啟加入好友介面

備份存檔偵測確保即使已完成當前房間，F5 仍會從房間入口重新載入，而非從房間完成後的狀態繼續。

### F6 — 單人多人模式切換

在主選單按 **F6** 切換**單人多人模式**，允許主機無需等待其他玩家加入即可開始多人對戰，適合單獨測試多人專屬內容。

## 多人功能狀態

| 功能 | 狀態 |
| --- | --- |
| 單人 F5 重新載入 | 已完成，已測試 |
| F6 單人多人模式切換 | 已完成，已測試 |
| 多人（主機） F5 重新載入 | 已完成，已測試 |
| 多人（其他玩家） F5 重新載入 | 已完成，已測試 |
