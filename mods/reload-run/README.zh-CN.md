# reload-run

[English](README.md) | 简体中文 | [繁體中文](README.zh-TW.md)

由 **Jayden Liang** 开发的 [Slay the Spire 2](https://store.steampowered.com/app/2868840/Slay_the_Spire_2/) mod。

## 功能

### F5 — 重载当前房间

在对局中按 **F5**，立即从当前房间的入口存档重新开始。

- **单人模式** — 原地重载，无需返回主菜单
- **多人（主机）** — 返回主菜单并自动以主机身份恢复已保存的多人对局
- **多人（其他玩家）** — 返回主菜单并自动打开加入好友界面

备份存档检测确保即使已完成当前房间，F5 仍会从房间入口重载，而非从房间完成后的状态继续。

### F6 — 单人多人模式切换

在主菜单按 **F6** 切换**单人多人模式**，允许 Host 无需等待其他玩家加入即可开始多人对局，适合单独测试多人专属内容。
